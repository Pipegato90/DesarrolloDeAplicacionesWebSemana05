# Galería Interactiva  
  
Una galería de imágenes interactiva creada con HTML, CSS y JavaScript.  
  
## Funcionalidades  
  
*   Agregar imágenes mediante una URL.  
*   Seleccionar una imagen para resaltarla.  
*   Eliminar la imagen seleccionada.  
*   Eliminar la imagen seleccionada con la tecla "Delete" o "Backspace".  
  
## Instrucciones de Uso  
  
1.  Ingresa la URL de una imagen en el campo de texto.  
2.  Haz clic en "Agregar Imagen".  
3.  Haz clic en una imagen para seleccionarla.  
4.  Haz clic en "Eliminar Imagen Seleccionada" o presiona la tecla "Delete" o "Backspace" para eliminar la imagen seleccionada.
